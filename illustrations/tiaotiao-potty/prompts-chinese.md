## 使用说明

### 在 Replicate 上生成
1. 访问 https://replicate.com/google/nano-banana-pro
2. 复制上方的 Prompt 粘贴到输入框
3. 设置 Aspect ratio 为 16:9
4. 点击 Generate 生成图片
5. 下载保存

### 使用 API 批量生成
```bash
export REPLICATE_API_TOKEN="你的API令牌"

# 图1
curl -X POST \
  -H "Authorization: Token $REPLICATE_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "version": "google/nano-banana-pro",
    "input": {
      "prompt": "Children'"'"'s book cover illustration...",
      "aspect_ratio": "16:9"
    }
  }' \
  https://api.replicate.com/v1/predictions
```

### 使用 baoyu-image-gen 工具
```bash
export REPLICATE_API_TOKEN="你的API令牌"

npx -y bun ~/.openclaw/workspace/skills/baoyu-image-gen/scripts/main.ts \
  --promptfiles prompts/01-scene-cover-hop.md \
  --image 01-scene-cover-hop.png \
  --provider replicate \
  --model google/nano-banana-pro \
  --ar 16:9
```
